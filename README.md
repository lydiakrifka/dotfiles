# Dotfiles

My Arch Linux configuration for Sound & Music Computing

## What's included

- **dwm** - Suckless window manager configuration
- **Neovim** - Editor setup with LSP support
- **zsh** - Shell configuration and aliases
- **kitty** - Terminal emulator config

## Installation
```bash
git clone https://github.com/lydiakrifka/dotfiles.git
cd dotfiles
```
